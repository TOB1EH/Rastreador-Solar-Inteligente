🎯 POR QUÉ USAR FZ0430:

1. PRECISIÓN GARANTIZADA (±0.1V)
   └─ Vs divisor de tensión (±0.5V)
   └─ 5x MEJOR precisión

2. PLUG & PLAY (Sin calibración manual)
   └─ No necesitas hacer cálculos especiales
   └─ Solo conectar 3 cables

3. RANGO AMPLIO (0-25V)
   └─ Puedes medir voltajes altos sin problemas
   └─ Divisor limitado a 0-3.3V en ESP32

4. COMPENSACIÓN TÉRMICA (Automática)
   └─ No varía con temperatura
   └─ Divisor necesita compensación manual

5. CÓDIGO SIMPLE
   └─ Menos líneas
   └─ Menos variables
   └─ Más limpio

🔧 CÓMO FUNCIONA:

Divisor de Tensión (Lo viejo):
┌────────────────────────────────────┐
│ Panel (5V)                         │
│    ↓                               │
│ [R1=10kΩ]                          │
│    ↓                               │
│ GPIO34 (mide 0-3.3V)               │
│    ↓                               │
│ Código ESP32:                      │
│ voltage = raw × 3.3 / 4095 × 2.0  │
│ (Necesita multiplicar por 2)       │
└────────────────────────────────────┘

FZ0430 (Lo nuevo):
┌────────────────────────────────────┐
│ Panel (5V)                         │
│    ↓                               │
│ [FZ0430 Módulo]                    │
│    ↓                               │
│ GPIO34 (mide 0-3.3V)               │
│    ↓                               │
│ Código ESP32:                      │
│ voltage = raw × 3.3 / 4095 × 5.0  │
│ (Solo multiplicar por 5)           │
└────────────────────────────────────┘

¡La diferencia es MÍNIMA en código, pero BIG en calidad!

⚡ CÓDIGO EJEMPLO (MUY SIMPLE):

// Sin FZ0430 (Divisor - complejo):
int adc_raw = adc_read(GPIO34);
float v_pin = adc_raw * 3.3 / 4095;
float v_panel = v_pin * 2.0;              // Multiplicar por ratio
if (temp > 25) v_panel *= 0.98;           // Compensación térmica
if (v_panel > 5.0) v_panel = 5.0;         // Limitar
if (v_panel < 0.0) v_panel = 0.0;

// Con FZ0430 (Plug & Play - simple):
int adc_raw = adc_read(GPIO34);
float v_pin = adc_raw * 3.3 / 4095;
float v_panel = v_pin * 5.0;              // Multiplicar por FZ0430
// ¡Ya está! No necesita más.
