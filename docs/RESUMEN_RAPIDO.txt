🎯 OBJETIVO:
─────────────────────────────────────────────────────────────────────────────
Crear un sistema autónomo que sigue la luz del sol rotando un panel solar
en eje horizontal (0-180°) usando un ESP32 + FreeRTOS, con almacenamiento
en cloud (Supabase) y control remoto por Telegram.

═════════════════════════════════════════════════════════════════════════════

📊 ESPECIFICACIONES CLAVE:
─────────────────────────────────────────────────────────────────────────────

CONTROLADOR:       ESP32 + FreeRTOS (multitarea)
SENSOR:            1x LDR (luz)
ACTUADOR:          1x Servo SG90 (0-180°)
PANEL SOLAR:       Inclinado 45°, rota horizontalmente
DISPLAY:           LCD 16x2 (métricas locales)
ALMACENAMIENTO:    Supabase (PostgreSQL cloud)
COMUNICACIÓN:      WiFi + Bot Telegram
AUTONOMÍA:         20+ horas (batería 18650)
EFICIENCIA:        +25-35% vs panel fijo

═════════════════════════════════════════════════════════════════════════════

🔧 LISTA DE HARDWARE:
─────────────────────────────────────────────────────────────────────────────

Controlador:           1x ESP32 DOIT DevKit v1
Motor:                 1x Servo SG90
Sensores:              1x LDR 5mm, 1x Resistor 10kΩ
Paneles:               2x Panel Solar 3V 100mA
Baterías:              2x Li-ion 18650 3000mAh
Cargador:              1x Módulo TP4056

Display:               1x LCD 16x2 + I2C backpack
Misc:                  Diodos, capacitores, cables, protoboard

COSTO ESTIMADO: $50-100 USD (componentes básicos)

═════════════════════════════════════════════════════════════════════════════

📱 ALGORITMO DE SEGUIMIENTO:
─────────────────────────────────────────────────────────────────────────────

1. Leer intensidad de luz (LDR) cada 100ms
2. Comparar con máximo conocido
3. Si luz baja 5%, activar búsqueda
4. Probar rotar +10° y medir luz
5. Si mejora, quedarse; si no, probar -10°
6. Guardar nueva posición óptima
7. Repetir desde paso 1

Resultado: Panel siempre orientado hacia máxima luz

🌐 SUPABASE:
─────────────────────────────────────────────────────────────────────────────

¿Qué es?  PostgreSQL cloud (como Firebase pero más potente)
Precio:   GRATIS (hasta 5GB)

Qué se guarda:
  • Timestamp, voltaje, corriente, potencia
  • Batería %, luz (ADC), ángulo servo, temperatura
  • Cada 10 segundos = 8.640 registros/día

Cómo acceder:
  1. Crear cuenta en https://supabase.com
  2. Crear proyecto (2 min)
  3. Copiar URL y ANON_KEY
  4. Ver datos en tiempo real en dashboard

📲 TELEGRAM BOT:
─────────────────────────────────────────────────────────────────────────────

Comandos disponibles:
  /status      ← Ver estado actual (voltaje, batería, luz, ángulo)
  /manual_0    ← Girar servo a 0° (Este)
  /manual_90   ← Girar servo a 90° (Centro)
  /manual_180  ← Girar servo a 180° (Oeste)
  /help        ← Mostrar ayuda

Control remoto: Desde cualquier lugar con tu móvil

❓ PREGUNTAS FRECUENTES:
─────────────────────────────────────────────────────────────────────────────

P: ¿Funciona sin conectar a WiFi?
R: Sí, el ESP32 funciona 100% autónomo. WiFi es para enviar datos a cloud
   y recibir comandos Telegram (opcional).

P: ¿Necesito PC conectada?
R: No. El ESP32 administra todo solo con FreeRTOS. La PC es solo para
   programar/compilar el código inicial.

P: ¿Cuánto cuesta Supabase?
R: Gratis hasta 5GB. Este proyecto usa ~5 MB/mes.

P: ¿Qué pasa si no tengo WiFi?
R: El sistema funciona perfecto. Solo no envía datos a cloud ni recibe
   comandos Telegram.

P: ¿Puedo controlar el servo desde Telegram?
R: Sí, con comandos: /manual_0, /manual_90, /manual_180

P: ¿Qué tan preciso es el seguimiento?
R: Error típico ±10° (más que suficiente para el sol)
